DEF CON 34 Machine-Readable Data

DEF CON 34 schedule data is published as static JSON and fetched by the web app at runtime. These JSON files are the authoritative sources for agents, crawlers, and search systems. Files may be replaced as conference information changes without rebuilding the site.

Primary Endpoints

  • Manifest JSON
  • Schedule JSON
  • Speaker and people JSON
  • Search index JSON
  • Menu JSON
  • Sitemap XML

Supporting Data

  • Locations JSON
  • Organizations JSON
  • Tags JSON
  • Events by day index JSON
  • Events by location index JSON
  • Events by person index JSON
  • Events by tag index JSON

Structure

Data lives under /ht/defcon34/. Entity files contain canonical records, index files map relationships for navigation, view files are precomputed lists for the client UI, and derived files provide app metadata such as the active menu.
